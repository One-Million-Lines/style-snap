# Security Policy

## Supported scope

Security fixes are handled on a best-effort basis for the current public code.

## Reporting a vulnerability

Please avoid public disclosure through issues for sensitive reports.

Use the maintainer contact path in `public.md` and include the affected area, reproduction steps, and impact.

## Security notes

- Uploaded images are handled in the browser for the current demo flow.
- No server-side image processing is included in this project.
- Keep any local `.env` files or private media files out of version control.
